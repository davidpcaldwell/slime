$loader

  • code
    string The code of the platform loader, as a string.
  • script
    Executes a JavaScript source file, overriding the default loader's implementation. Should execute the script(Scriptable,String,InputStream) method of the current inonit.script.rhino.Engine object.
    Arguments
    1. object A scope into which to execute the file.
    2. string A name to use for the file, for the purposes of debugging information, stack traces, etc.
    3. JavaClass java.io.InputStream An input stream from which the file can be read.

Exports

  • module
    Invokes the platform loader's module method.
    Arguments
    1. JavaClass inonit.script.rhino.Module A module to load.
    2. The second argument to the platform loader's module method.
    Returns
    The return value of the platform loader's module method.
  • script
    Invokes the platform loader's script method.
    Arguments
    1. Either the first argument to the platform loader's script method or an object describing a code file.
      The first argument to the platform loader's script method,
      OR
      An object describing a code file.
      has properties:
      • name
        string A name to use for this file, for debugging, stack traces, etc.
      • $in
        JavaClass java.io.InputStream An input stream from which the file can be read.
    2. The second argument to the platform loader's script method.
    Returns
    The return value of the platform loader's script method.
  • namespace
