$loader

  • code
    string The code of the platform loader, as a string.
  • script
    Executes a JavaScript source file, overriding the default loader's implementation.
    Under the current Rhino embedding design, this method should execute the script(String,InputStream,Scriptable,Scriptable) method of the current inonit.script.rhino.Engine object.
    Arguments
    1. string A name to use for the file, for the purposes of debugging information, stack traces, etc.
    2. JavaClass java.io.InputStream An input stream from which the file can be read.
    3. object A scope into which to execute the file.
    4. object A target object to use as this when executing the file.

Exports
