The Rhino SLIME loader contains a bundle of Java and JavaScript code that is knitted together at runtime. Users of the Rhino loader can create one inside a Java host object and provide it to scripts. Inside the Java host object, an instance of inonit.script.rhino.Engine must be created, and then inonit.script.rhino.Loader.load(inonit.script.rhino.Engine,inonit.script.rhino.Loader) must be invoked with an instance of Loader that knows how to locate the platform loader code (loader/literal.js) and the Rhino loader script (loader/rhino/literal.js).
