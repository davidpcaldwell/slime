This page contains resources for developing SLIME itself.

Tests

  • jsh Integration tests: jsh/etc/integration.jsh.js -part part
  • Browser module tests
    • Run all browser module tests: loader/browser/etc/unit.jsh.js
    • Run selected browser module tests: jsh/unit/browser.jsh.js [-interactive] file [file...]
      Example: jsh/unit/browser.jsh.js loader/\$api.js
      Arguments:
      • -interactive: Do not close browser on exit.
      • -chrome, -firefox, -safari, -ie: Run tests in specified browser. If no browsers are specified, all detected browsers will be run.
      • -chrome:profile pathname: Use the specified location for storing Chrome's user profiles; useful for having multiple Chrome environments configured for local development.
