SLIME WebView application run inside an embedded JavaFX browser derived from WebKit.

SLIME Browser

The browser is provided with an implementation of alert that routes messages back to the containing program.

webview.initialize.js

If the page inside the browser executes a script named window.initialize.js, the page will conditionally load additional features to allow enhanced communication between the page and the embedding process. This script can safely be used in an application that is compatible with running inside or outside WebView, as loading it in an ordinary browser will have no effect.

window.console

The browser is provided with a console that routes messages back to the containing program.
