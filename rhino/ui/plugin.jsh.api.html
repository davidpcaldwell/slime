The jsh.ui package extends the functionality available in the UI module. The UI module is configured as follows:
  • JavaFX is enabled if the javafx.embed.swing.JFXPanel Java class is present. In practice, this currently limits JavaFX usage to Java 1.8 and higher.
  • When an application exits, java.lang.System.exit() is called.
