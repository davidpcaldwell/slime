The api.js script configures a local jrunscript execution environment to contain APIs that are useful for jrunscript-based scripts. Depending on how it is invoked, it may also execute other programs.

Context

The script is typically invoked with the global object as this, although it is compatible with using any object for this purpose.
It can receive parameters if they are already attached to this:
  • $api
    An object providing configuration information to the script. This object will be overwritten by the script itself.
