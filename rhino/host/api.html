  • invoke
    Invokes a method on a Java object via reflection. Ordinarily, Java methods can simply be invoked with JavaScript semantics using a LiveConnect implementation. The invoke function provides the ability to invoke methods that are not accessible (for example, those that are private, package or protected).
    Arguments
    1. has properties:
      • target
        Packages.java.lang.Object The Java object on which to invoke the method. May be omitted for static methods.
      • method
        An object specifying the Java method to invoke.
        has properties:
        • name
          string The name of the method to invoke.
        • parameterTypes
          Array of JavaClass (optional; defaults to a method with no parameters) The list of parameter types in the signature of the method to invoke.
        • class
          JavaClass The declaring class of the method to be invoked. If the method is declared in the concrete class of the object, this value may be omitted.
      • arguments
        Array of Packages.java.lang.Object (may be omitted if the Java method takes no arguments) The arguments with which to invoke the method.
    Returns
    Packages.java.lang.Object The value returned by the Java method, or undefined for void methods.
