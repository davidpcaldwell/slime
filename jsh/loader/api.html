  • module
    Invokes the Rhino loader's module method.
    Arguments
    1. jsh.file.Pathname The location of the module. If the file is a directory, it is treated as the base of an unpacked module. If the file is a plain file, then if its name ends in .slime, it is treated as a packed module. Otherwise, the given file is treated as the main file of a module whose base is the directory in which the file is located.
    2. object (optional) The object to use as the $context object for this module; if not specified, an empty object will be supplied.
    Returns
    The return value of the Rhino loader's module method.
  • namespace
    Invokes the Rhino loader's namespace method.
  • addClasses
    Adds a directory or JAR file to the Java classpath of this script.
    Arguments
    1. jsh.file.Pathname A jsh.file.Pathname pointing to a directory or a JAR file containing Java classes.
