jsh/tools/install/rhino.jsh.js

Installs Rhino into the shell used to run the script.

Command-line arguments

  • -local pathname: Specifies the location of a local copy of the Rhino js.jar library.
  • -replace: Causes the installation to proceed even if Rhino is found.
