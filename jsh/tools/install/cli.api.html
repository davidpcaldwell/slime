jsh/tools/install/rhino.jsh.js

Installs Rhino into the shell used to run the script.

Command-line arguments

  • -local pathname: Specifies the location of a local copy of the Rhino js.jar library.
  • -replace: Causes the installation to proceed even if Rhino is found.

jsh/tools/install/ncdbg.jsh.js

Installs the ncdbg debugger for debugging Nashorn scripts.

Usage

  • -version: specifies the version to install. Permissible values presently are:
    • 0.8.0: installs the 0.8.0 build
    • master: installs from source, using the master version from a fork.
    • 0.8.1: installs the 0.8.1 build
  • -replace: specifies that the implementation should be replaced if found.
