Settings

Name Description
jsh.debug.jdwp Sets up a Java debugger to debug the shell. The value of this setting is interpreted as an argument to the -agentlib-jdwp: Java launcher argument.
jsh.jvm.options A set of literal options to be passed to the Java launcher when starting the Java virtual machine running the shell.
jsh.log.java.properties Specifies the location of a java.util.logging properties file to use in configuring logging for the Java virtual machine running the shell.
