Java launcher

When jsh is built, its installation directory contains the file jsh.jar, containing the classes necessary to start a jsh shell. So jsh can be invoked with:
java -jar jsh.jar <script> [arguments]

Environment variables

Several environment variables affect a jsh shell.
  • JSH_SCRIPT_DEBUGGER: if set to profiler or a value starting with profiler:, and a shell including a profiler is being run, the loader process will be run under a profiler. If set to rhino, the main script will be run under the Rhino debugger. The profiler may be configured with an argument like: profiler:name1=value1,name2=value2.
