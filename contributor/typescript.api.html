TypeScript in SLIME

TypeScript runtime support

TypeScript can be supported in SLIME embeddings by providing a typescript property of type slime.runtime.TypeScript on the $slime object provided to the runtime.

jsh support

The implementation in inonit.script.jsh.Shell essentially looks for the Node installation in the jsh lib/ directory and creates a Loader.TypeScript that runs tsc there if it exists.

Testing

See jsh/tools/test/typescript.jsh.js.

How to declare type metadata for TypeScript in SLIME code

This document describes tested, working practices for defining TypeScript-compatible type information within SLIME itself.
See the TypeScript documentation on using JSDoc to specify TypeScript information for details.
